# 람다

람다식의 형식 
>매개변수가 있는 경우 : `(매개변수 목록) ->  {실행 코드 블럭}`
>매개변수가 없는 경우 : `() ->  {실행 코드 블럭}`

람다식 문법
- 실행 코드 블록이 한 줄인 경우 중괄호 `{}` 생략 가능하다. 
- 매개변수의 타입이 추론 가능하면 생략 가능하다. 
- 단일 매개변수인 경우 소괄호 `()` 생략 가능하다. 
- 실행 코드 블록이 반환값을 가질 경우 `return` 생략 가능하다. 

사용법 
- 메서드의 매개변수로 전달할 수 있다.
- 함수형 인터페이스 구현시 사용 
  > 단 하나의 추상 메서드만을 가진 인터페이스를 함수형 인터페이스라 한다. 
- 스트림 API 에서 자주 사용된다.
