> Collection을 포함한 클래스는 반드시 다른 멤버 변수가 없어야 한다.
   ( Collection을 Wrapping 하면서, Wrapping한 Collection 외 다른 멤버 변수가 없는 상태 = 일급 컬렉션 ) 

Collection 의 Wrapping 한다는 것은 무엇인가? 
```java
public class Person {
    private String name;
    private List<Car> cars;
}

public class Car {
    private String name;
    private String oil;
}
```

위 코드를 아래와 같이 바꾸는 것 

```java
public class Person {
    private String name;
    private Cars cars;
}

// List<Car> cars를 Wrapping
// 일급 컬렉션
public class Cars {
    // 멤버변수가 하나 밖에 없다!!
    private List<Car> cars;
}

public class Car {
    private String name;
    private String oil;
}
```
